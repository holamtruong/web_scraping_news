# NEWS APP
## Setup environment

### Install flask
    >>> pip install flask

### Install newspaper3k
    >>> pip install newspaper3k

### Clear DB (execute SQL)
    delete from news;    
    from sqlite_sequence where name='news';

## How to use:
- Run 'utils.py' to get and insert article.

- Run 'index.py' to start server.

- Run 'run.bat' to start server.
